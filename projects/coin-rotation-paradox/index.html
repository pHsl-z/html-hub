为什么是 1 周？

内滚时小圆顺时针自转，公转逆时针，方向相反：

$$\text{自转圈数} = \frac{R}{r} - 1$$

当 R/r = 2 时：2 − 1 = 1 周

内摆线方程：

$$x = (R-r)\cos\theta + r\cos\!\left(\tfrac{R}{r}-1\right)\theta$$
$$y = (R-r)\sin\theta - r\sin\!\left(\tfrac{R}{r}-1\right)\theta$$

当 R=2r 时，y 项消去，内摆线退化为一条直径。

为什么是 3 周？

外滚时自转方向与公转方向相同，两者叠加。

$$\text{自转圈数} = \frac{R}{r} + 1$$

当 R/r = 2 时：2 + 1 = 3 周

外摆线方程：

$$x = (R+r)\cos\theta - r\cos\!\left(\tfrac{R}{r}+1\right)\theta$$
$$y = (R+r)\sin\theta - r\sin\!\left(\tfrac{R}{r}+1\right)\theta$$

圆心路程 = 2π(R+r) = 6πr = 3×小圆周长 → 自转 3 周。直觉认为 2 周，实际 3 周——这就是著名的硬币旋转悖论。
